# Week 2 — Static HTML Portfolio

## 🎯 Objective

Build a multi-page **static personal portfolio website** using plain HTML5, demonstrating basic web structure, navigation, and content layout — without any CSS frameworks or JavaScript.

---

## 📁 Files

| File | Description |
|------|-------------|
| `index.html` | Home page — student name, roll number, and brief introduction |
| `academics.html` | Academic marks/results displayed in an HTML table |
| `personal.html` | Personal details page |
| `contact.html` | Contact information page |
| `resume.html` | Resume/CV page |
| `resume.pdf` | Downloadable PDF resume |
| `Image1.png`, `Image2.png` | Images used in the pages |

---

## 📖 Concepts Covered

- HTML document structure (`<!DOCTYPE>`, `<head>`, `<body>`)
- Navigation using anchor (`<a>`) tags for multi-page linking
- HTML tables (`<table>`, `<tr>`, `<th>`, `<td>`) for structured data
- Semantic layout using `<div>` blocks for header, main, and footer sections
- Embedding images with `<img>`

---

## 🌐 How to View

Since these are static HTML files, no server is needed. Simply open `index.html` in any web browser:

```bash
# From inside the Week 2 folder:
open index.html       # macOS
start index.html      # Windows
xdg-open index.html   # Linux
```

Or drag and drop `index.html` into your browser window.
